Klasifikasi Analisis Sentimen Movie Review Dengan Metode Support Vector Machine

Menggunakan Kernel Radial Basis Function dan Information Gain

Petunjuk Penggunaan

1. Siapkan dataset review dengan format .csv yang terdiri dari dua kolom. Kolom pertama untuk review berikan nama "Message" dan kedua untuk sentimen berikan nama "kelas"
2. Pilih menu Dataset untuk import data review yang akan digunakan atau untuk mengganti dataset baru
3. Lakukan text pre-processing data dengan memilih tombol "Text Pre-processing" pada halaman Dataset
4. Lakukan pembobotan fitur dengan memilih tombol "Information Gain" pada halaman Text Pre-processing
5. Lakukan pengujian klasifikasi analisis sentimen dengan algoritma SVM pada menu "Klasifikasi SVM"
6. Hasil akurasi & confusion matrix akan ditampilkan
7. Lakukan pengujian klasifikasi analisis sentimen dengan algoritma SVM kernel RBF pada menu "Klasifikasi SVM RBF"
8. Masukkan nilai paramter C dan gamma terlebih dahulu, setelah itu pilih "Process"
9. Hasil akurasi & confusion matrix akan ditampilkan
10. Lakukan pengujian klasifikasi analisis sentimen dengan algoritma SVM kernel RBF dan IG pada menu "Klasifikasi SVM RBF IG"
11. Masukkan nilai paramter C, gamma dan jumlah top k (rentang 1-4999 fitur) terlebih dahulu, setelah itu pilih "Process"
12. Hasil akurasi & confusion matrix akan ditampilkan
